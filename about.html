1/ Le projet :

1.1/ Pourquoi ?

Ce projet est né de plusieurs frustrations :

    1/ Avoir une fenêtre ouverte pour écouter sa musique est insupportable lorqu'on travaille.

    2/ Etant freelance et digital nomad, je n'ai pas toujours accès au wifi, je voulais pouvoir écouter ma musique sans y avoir accès.

    3/ Je n'aime pas être dérangé par les pubs de Youtube ou des autres plateformes.

1.2/ Ce que j'ai appris :

    1/ Créer une extension Web

    2/ Faire tourner du code en background de chrome

    3/ Implémenter la "REST API" de Airtable

    4/ Utiliser JS pour afficher du contenu dans des pages de manière dynamique

    5/ Améliorer mon niveau de script


    Ce projet n'a aucune ambition lucrative, je l'ai réalisé dans l'unique but d'apprendre et de rendre service.
